# webpack_learn

## webpack 学习案例代码


`下载代码`
```js
git clone https://github.com/VuePack/webpack_learn.git
```

`安装依赖`
```bash
npm install
```

`运行`
```bash
npm start
```

`编译`
```bash
webpack
```
